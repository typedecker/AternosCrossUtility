# AternosCrossUtility Plugin
A browser based plugin that allows for automatic confirmation of queue waiting and instant click on the adblocker overlay skip button. Supports all major browsers Chromium(Google Chrome, Opera, Opera GX, Brave, Edge) + Firefox(All forms).

## Installation Instructions(For Chromium Browsers)

1. Download this repo as a .zip by clicking on the button shown in the image below.
   <img width="728" height="328" alt="image" src="https://github.com/user-attachments/assets/41d2af5f-a5cf-4107-862d-4e21f9cb7b35" />
2. Extract the .zip file into a folder.
3. Go to `chrome://extensions` or `edge://extensions` or `opera://extensions` or `brave://extensions` depending on the browser you're installing it into.
4. Enable developer mode from the top right, and then click on Load unpacked as shown in the image below. [the overall method may remain the same, but the UI and location of toggles may be different depending on the browser used]
   <img width="1308" height="130" alt="image" src="https://github.com/user-attachments/assets/5eac0898-b0c4-4e7a-9b2d-117b4f18f9cb" />
5. On clicking the load unpacked button, a folder selection dialog will be opened, choose the unzipped folder and select it.
6. Enable the extension if you haven't by finding it in the list of extensions and toggling the switch to an ON state.
7. Thats it! Now you can enjoy using the extension to make your life easier when dealing with aternos <3!

1. Copy the code inside script.js in this repo. You can do so, by clicking on script.js in the content list and then pressing the copy button as demonstrated in the image below.

   <img width="716" height="308" alt="image" src="https://github.com/user-attachments/assets/e2310c55-48e0-4b75-97f2-a233c53f26e2" />
   <img width="1280" height="588" alt="image" src="https://github.com/user-attachments/assets/cdaa24b0-084a-42f9-8f7c-09db120fea97" />

3. Download the violentmonkey firefox plugin from the following link: https://violentmonkey.github.io/get-it/
4. After installation of violentmonkey, create a new script as demonstrated in the image below, and paste the code copied from script.js, into the script area.

   <img width="333" height="166" alt="image" src="https://github.com/user-attachments/assets/9aef69fe-33d4-4df5-a7b2-621d494f5f8b" />
5. Make sure the scripts are enabled, and now you have a permanent way of running this plugin using a script instead!
6. Thats it! Now you can also enjoy using the extension via this script, to make your life easier when dealing with aternos <3!
